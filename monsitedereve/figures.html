Les figures

1) L-Sit

Description : En appui sur les mains, jambes tendues à 90°.

Travaille : abdos, fléchisseurs de hanches, épaules.

Objectif : tenir 10–30 secondes.

2) Tuck Planche

Description : Version débutante de la planche, genoux ramenés à la poitrine.

Travaille : épaules, triceps, gainage intense.

Objectif : apprendre l’équilibre vers l’avant.

3) Front Lever

Description : Suspendu à une barre, corps à l’horizontale.

Travaille : dos, bras.

Difficulté : demande une forte traction.

4) Back Lever

Description : Suspendu, corps horizontal face vers le sol.

Travaille : épaules, dos, biceps.

5) Planche

Description : Corps parallèle au sol, uniquement en appui sur les mains.

Travaille : épaules, triceps, abdos, gainage total.

6) Human Flag

Description : Corps horizontal tenu sur un poteau vertical.

Travaille : obliques, épaules.
